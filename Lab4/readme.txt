Rješenje četvrte domaće zadaće iz predmeta APR

Zadaća je napisana u je programskom jeziku Python 3.8.
Svi zadaci se pokreću s naredbom: python main.py
Ta skripta poziva sve zadatke i pokreće ih.
Ona koristi klase iz skripti klasa_funkcije.py, klasa_postupci.py, klasa_ogranicenja.py.

Za svaki postupak moguće je učitati ulazne parametre iz datoteke. 
Za to je potrebno poslati u funkciju ime datoteke.
